# TODOLIST
This web application allow you to add delete in the list. And allow you to make different lists

# Tech Stack
 <ul>
  <li>Node.js</li>
  <li>Express.js</li>
  <li>MongoDB</li>
  </ul>

## For running

- Run npm install
- install mongoose
- create env file and add address or localhost for DBURL to set up the connection
